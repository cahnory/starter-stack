## Install

### Packages

  + [babel-preset-react](babel-preset-react)

#### babel-preset-react

Make babel support jsx syntax. [read more](http://babeljs.io/docs/plugins/preset-react/)

### Config

#### .babelrc

Tel babel to use the react preset.

```json
{
  "presets": [
    "react"
  ]
}
```
