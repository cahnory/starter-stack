# Babel

## Usage

### Scripts

| name        | command                   |
|-------------|---------------------------|
| build       | `babel -s -d dist src`    |
| watch       | `babel -w -s -d dist src` |


## Install

### Packages

  + babel-cli
  + babel-core
  + babel-preset-es2015
  + babel-preset-stage-0

### Config

#### .babelrc

```json
{
  "presets": [
    "es2015",
    "stage-0"
  ]
}
```
